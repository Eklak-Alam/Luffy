# Eklak Alam — Software Engineer & Cloud Architect Portfolio

> A premium, highly interactive personal portfolio built to showcase scalable architectures, cloud infrastructure expertise, and high-end user experiences. 

[![Live Demo](https://img.shields.io/badge/Live_Site-eklak.site-609966?style=for-the-badge)](https://eklak.site)

## 🌐 Overview

This repository contains the source code for my personal portfolio. The site is designed with a heavy focus on fluid physics-based animations, modern typography, and a mobile-first responsive layout. It serves as the central hub for my project case studies, professional experience, and direct contact.

## ✨ Core Features

* **Physics-Based Animations:** Complex, timeline-driven scroll animations using **GSAP** and **Framer Motion** for a buttery-smooth user experience.
* **Custom Mobile Navigation:** A scratch-built, zero-scroll lock mobile menu with a custom morphing hamburger toggle and animated layout indicators.
* **Serverless Contact Transmission:** Fully functional, client-side validated contact form powered by **EmailJS**, requiring no backend to send inquiries directly to my inbox.
* **Advanced SEO & Open Graph:** Fully optimized metadata, canonical URLs, and Twitter/LinkedIn card integration to ensure maximum visibility and professional link sharing.
* **Custom Theming:** A bespoke "Architect" color palette mapped through Tailwind CSS variables, complete with custom selection colors and minimalist scrollbars.

## 🛠️ Tech Stack

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Library:** [React](https://reactjs.org/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Animation:** [GSAP](https://gsap.com/) & [Framer Motion](https://www.framer.com/motion/)
* **Icons:** [Lucide React](https://lucide.dev/)
* **Email Service:** [EmailJS](https://www.emailjs.com/)
* **Typography:** Geist & Geist Mono (next/font)
* **Deployment:** [Vercel](https://vercel.com/)
